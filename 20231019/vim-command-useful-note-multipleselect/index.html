Vim 同時操作多行或多個相同單字的方法

記錄一下用 Vim 怎麼選取並同時編輯多行？還有怎麼選取多個相同單字並修改？
因為懶得輸入太長的指令，喜歡按多個組合鍵 combo ，所以以下的方法大家可以參考就看看就好喔！

情境整理

一次選取並編輯多行內容

  1. 先在 Normal Mode 中選取要編輯的多行，這邊我使用的是 V 加上 j 往下跳到欲選取的最後一行。

  2. 選取完畢之後，使用 A 或是 I 後就可以編輯了。

同時選取並編輯多個相同單字

  1. 先在 Normal Mode 中選取該單字，我使用的是 viw 選取游標停留位置上的單字。

  2. 使用 command + d 的方式選取多個相同的單字。

  3. 最後使用 A 或是 I 後就可以編輯了。
